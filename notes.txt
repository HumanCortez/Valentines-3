# Valentines-3

## how to open it

1. find the file called `index.html` in this folder
2. double click it
3. it should open in your browser
4. if it does not, right click the file and choose “Open With” then pick Chrome, Safari, or any browser
5. if sound does not start, tap any button once
6. if it says locked, it just means it is not midnight yet

that is it
